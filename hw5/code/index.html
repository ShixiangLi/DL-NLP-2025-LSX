交通信息

米兰拥有高效便捷的公共交通系统，主要由ATM公司运营。

地铁 (Metropolitana)

米兰地铁目前有5条线路（M1, M2, M3, M4, M5），覆盖城市主要区域和机场。是游览城市最快捷的方式之一。可在报刊亭、烟草店或自动售票机购买车票。

有轨电车 (Tram) & 公交 (Bus)

米兰的有轨电车和公交网络非常发达，可以带您到达地铁无法直接覆盖的区域。部分老式有轨电车本身就是一道风景线。车票与地铁通用。

机场交通

主要机场：马尔彭萨(MXP)、利纳特(LIN)、贝加莫(BGY)。
- 马尔彭萨机场(MXP): 可乘坐机场快线火车(Malpensa Express)直达市中心，或选择机场大巴。
- 利纳特机场(LIN): 距离市区较近，可乘坐公交73路或机场大巴。
- 贝加莫机场(BGY): 主要为廉航服务，可乘坐机场大巴前往米兰中央火车站。

自行车共享 (BikeMi)

BikeMi是米兰的公共自行车租赁服务，站点遍布城市。适合短途骑行，体验城市慢生活。需要提前注册。

美食推荐

米兰不仅是时尚之都，也是美食天堂。别错过这些经典美味：

米兰烩饭 (Risotto alla Milanese)

金黄色的藏红花烩饭，香气浓郁，是米兰的标志性菜肴。

米兰炸小牛排 (Cotoletta alla Milanese)

带骨的炸小牛排，外酥里嫩，简单却美味。

炖小牛膝 (Ossobuco)

通常与米兰烩饭搭配，小牛膝用白葡萄酒和蔬菜慢炖而成，肉质鲜美。

潘妮朵尼 (Panettone)

源自米兰的圣诞节传统甜面包，松软香甜，带有水果干。

意式冰淇淋 (Gelato)

在米兰街头，随处可见美味的Gelateria，一定要品尝各种口味的Gelato。

开胃酒时光 (Aperitivo)

傍晚时分，在酒吧点一杯饮品，即可享用免费的自助小食。纳维利区 (Navigli) 是体验Aperitivo的好去处。

探索更多：Trattoria (传统家庭式餐馆), Ristorante (正式餐厅), Pizzeria (披萨店)。

住宿选择

米兰提供多样化的住宿选择，从奢华酒店到经济型旅馆应有尽有。

豪华酒店

集中在黄金四角区及大教堂附近，提供顶级设施和服务。

精品酒店与B&B

在布雷拉区 (Brera) 或纳维利区 (Navigli) 可以找到许多设计独特、温馨舒适的精品酒店和B&B。

公寓租赁

适合家庭或长期停留的游客，提供更居家式的体验。可通过各大在线平台预订。

经济型住宿

中央火车站 (Stazione Centrale) 附近有较多经济型酒店和青年旅社，交通便利。

建议提前通过 Booking.com, Expedia, Airbnb 等平台预订，尤其是在旅游旺季和时装周期间。

热门景点

米兰大教堂 (Duomo)

米兰的象征，宏伟的哥特式教堂。可登顶俯瞰城市风光。

埃马努埃莱二世长廊

历史悠久的奢华购物拱廊，连接大教堂广场和斯卡拉广场。

斯福尔扎城堡

曾是米兰统治者的居所，现为博物馆群，藏有达芬奇和米开朗基罗的作品。

《最后的晚餐》

达芬奇的旷世名作，位于圣玛利亚感恩教堂。务必提前数月预订门票。

布雷拉美术馆

意大利最重要的美术馆之一，收藏了众多文艺复兴时期及巴洛克艺术杰作。

纳维利运河区 (Navigli)

古老的运河区，傍晚时分尤其热闹，酒吧、餐厅林立，是体验Aperitivo的好地方。

实用攻略

安全提示

米兰总体安全，但在人多拥挤的景点（如大教堂广场、地铁）和中央火车站附近需注意小偷和扒手。保管好个人财物。紧急电话：112。

货币与支付

意大利使用欧元 (€)。信用卡和借记卡广泛接受，但建议随身携带少量现金以备不时之需（如小额消费、部分小店）。

语言

官方语言为意大利语。旅游区和酒店通常可以说英语。学几句简单的意大利语会更受欢迎：
你好 (Ciao), 谢谢 (Grazie), 请 (Per favore), 是 (Sì), 不是 (No), 对不起/请问 (Scusa/Mi scusi)。

小费

在意大利，小费不是强制性的，服务费通常已包含在账单中 (coperto 或 servizio)。如果服务特别好，可以额外给一些小费表示感谢 (例如账单金额的5-10%或凑整)。

电源插座

意大利标准电压为230V，频率为50Hz。插座类型为Type F (双圆孔) 和 Type L (三圆孔)。请根据需要准备转换插头。

购物与退税

米兰是购物天堂。非欧盟居民在符合条件的商店购物达到一定金额可申请退税。购物时请向店家索取退税单 (Tax Free Form)。

活动日历

米兰全年都有精彩纷呈的活动。以下是一些主要的年度盛事：

米兰时装周

每年两次 (2-3月秋冬系列, 9-10月春夏系列)，全球时尚界的顶级盛事。

米兰设计周 (Salone del Mobile)

通常在每年4月举行，是全球规模最大、最重要的家具与设计展览。

圣诞集市与新年

12月起，米兰会有多个圣诞集市，大教堂广场的新年庆祝活动也十分热闹。

具体的活动日期和更多信息，请查询米兰旅游局官方网站或当地活动指南。
